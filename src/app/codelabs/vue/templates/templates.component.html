Templates / Intro

Vue has a very straightforward template system, which extends HTML with a reactive DOM to make it dynamic.

Reactive: Changes in data emitted by one component will flow through the app to any other components listening for those changes.

Templates / Interpolation

You can interpolate data from your Vue app into the template using double curly braces.

Backticks ` ` are magic quotes that allow multi-line strings and text interpolation.

Templates / Interpolation

The double curlies can also contain JavaScript expressions. For example, you can evaluate a Vue instance method (like printHello() below), or calculate 323213 + 34234

Templates / Property Binding

You can also make HTML attributes reactive with Vue's bind directive: v-bind:attribute="property"

v-bind can also take expressions.
v-bind: can be replaced with its colon shorthand, e.g. :attribute="property".

Templates / Event binding: (event)

v-bind passes data from the Vue app to the template, but what if you wanted to flow data in the other direction?

v-on: can be replaced with its commercial at shorthand, e.g. @event="method()".
