What is JSX?

The above syntax is not HTML, it's called JSX. It is an extension for JavaScript, not a template engine. It's transformed to JavaScript with a compiler like babel or TypeScript.

What is JSX?

Fundamentally, JSX just provides syntactic sugar for the React.createElement function. It's important to remember that you are not writing any HTML in React. We will learn about JSX syntax later in this course.

Rendering React

You can transform React components to DOM elements in browsers, native controls in mobile apps, procedures for hardware, sounds and music, games, whatever. All these tools actually exist.


A tool that trasnforms (or renders) React components for a specific medium is called renderer.


To use React in browser we need DOM renderer, which comes as a separate package called ReactDOM.

ReactDOM

ReactDOM provides a render() function which receives a root React component (remember that UI is a tree?) and a DOM element. Then it mounts the component to the element.
