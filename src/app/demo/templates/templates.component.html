Angular Templates - Data binding extras

Angular supports more advanced bindings than just attribute name.

Event binding ( (event) )

The bindings directives you've met so far flow data in one direction: from a component to an element.

Conditional Display (*ngIf)

*ngIf renders a template on the DOM when its assigned expression evaluates to truthy.

Reference binding ( #var )

Sometimes we need to access a component from the template. We can mark it with #name, and it because available as name everywhere in the template.

Conditional Display (*ngFor)

Loops over items in a collection and adds a template to the DOM for each item in the collection.

The *ngFor directive will display one puppy name for every puppy in the list.

Note the * that identifies a DOM structure-altering directive and the camelCase!
