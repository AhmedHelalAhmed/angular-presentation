TypeScript

TypeScript is a language that compiles to JavaScript

  • Created by Anders Hejlsberg, lead architect of Turbo Pascal, Delphi and later C#
  • Rather than try to replace JavaScript, TypeScript embraces and extends it.
  • TypeScript adds an optional Type System as well as features from the latest versions of JavaScript not yet supported in all browsers otherwise
